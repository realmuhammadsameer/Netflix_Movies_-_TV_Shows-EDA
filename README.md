# Netflix EDA Project

## Objective
Perform Exploratory Data Analysis (EDA) on the Netflix dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Dataset Information (.info())
- Statistical Summary (.describe())
- Value Counts
- Histogram
- Boxplot
- Scatterplot
- Heatmap
- Pairplot

## Key Findings
- Movies are more common than TV Shows.
- Most content was released after 2010.
- Numerical variables show weak correlations.
- A few outliers are present in the dataset.
